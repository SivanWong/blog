双飞翼布局

双飞翼布局和圣杯布局几乎一样，区别在于处理middle中被遮挡的部分。

  • 圣杯布局是在三栏布局的父容器中设置padding。
  • 双飞翼布局是在middle中再放一个div用来显示内容，为其设置margin。
