### toLocaleString方法

```
const num = 12345678;
num.toLocaleString();
```

toLacalString：返回这个数字在特定语言环境下的表示字符串。

在没有指定区域的基本使用时，返回使用默认的语言环境和默认选项格式化的字符串。

### 正则表达式

```
const num = 12345678;
var reg=/\d{1,3}(?=(\d{3})+$)/g;
num.replace(reg,"$&,");

1. 正则表达式 \d{1,3}(?=(\d{3})+$)  表示前面有1~3个数字，后面的至少由一组3个数字结尾。
2. ?=表示正向引用，可以作为匹配的条件，但匹配到的内容不获取，并且作为下一次查询的开始。
3. $& 表示与正则表达式相匹配的内容。
```
