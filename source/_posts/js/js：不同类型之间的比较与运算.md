---

```
1. 5+'5' // '55'
2. 5-true // 4
3. 5-'2' // 3
4. '21'>3 // true
5. null===undefined // false
6. NaN===NaN // false
```
- 无论字符串与什么做+运算，最终结果都是拼接而成的字符串
- 无论数字与什么做-运算，另一方都会被转换成数字进行运算，结果仍为数字
- 字符串与数字做比较时总会把字符串转换为数字进行比较，当字符串无法转换为数字时，结果总为false
- ===会进行类型比较，不会进行转换，其余比较均会
